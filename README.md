# ETL-LLM-for-Weather-Processing
## Installation
### 2. Crear el entorno virtual

```bash
python -m venv .venv
```

### 3. Activar el entorno virtual

**En PowerShell:**

```powershell
.\.venv\Scripts\Activate.ps1
```

**En CMD:**

```bat
.\.venv\Scripts\activate.bat
```

### 4. Instalar dependencias

```bash
pip install -r requirements.txt
```

### 5. Desactivar el entorno virtual

```bash
deactivate
```

We are using 3 meteorogical APIs Openwheatermap, wheatherapi and open-meteo from this 3 open-meteo is the only one that doesnt need an API key for the free plan.
